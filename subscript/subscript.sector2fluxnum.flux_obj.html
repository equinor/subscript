include_nnc(egrid_file)

Adds NNC connections to FLUXNUM region.

@EGRID file

This is useful for LGR since LGR region must be one cell away from FLUXNUM boundaries. If NNC for LGR is outside FLUXNUM, an error will be sent by ECLIPSE.

Needs NNC data from EGRID file.

include_well_completions(completion_list, well_list, exclude_list=None)

Includes well completions in FLUXNUM keyword. This is necessary for ECLIPSE to accept the FLUXNUM region.

@completion_list : List of completions to be included. @well_list : List of wells to be included @exclude_list = List of wells to be excluded from the FLUXNUM. Usually empty.
