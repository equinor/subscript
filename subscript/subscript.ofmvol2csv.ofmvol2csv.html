subscript.ofmvol2csv.ofmvol2csv.process_volstr(volstr)

Parse a volstring (typically a vol-file read into a string with newline characters)

Two different data syntaxes are supported. Either each well is given in separate blocks of lines, then there is a *NAME line that gives the well name.

The alternative syntax has wellname encoded in the table as any other parameter, in the *WELL column.

Parameters:

volstr (str)

Return type:

DataFrame

Returns:

Dataframe indexed by WELL and DATE.

subscript.ofmvol2csv.ofmvol2csv.split_list(linelist, splitidxs)

Split a list of lines into chunks, where each chunck is a list of lines with each chunk only containing data for one well

Example:

split_list(['a', 'b', 'c', 'd', 'e', 'f'], [2,5])

would return:

[['a', 'b'], ['c', 'd', 'e'], ['f']]
Parameters:

  • linelis – List to divide into chunks.

  • splitidx – Indexes (int) at which to split, the index points to the left edge of the resulting chunk. Repeated indices are ignored.

Return type:

list[list[str]]

Returns:

list of lists of strings
