CONVERT_GRID_FORMAT

Convert between various 3D grid cornerpoint formats

usage: convert_grid_format ...

Named Arguments

--conversion

Conversion method, select from [‘ecl2roff’] (default ecl2roff)

Default: “ecl2roff”

--mode

Mode: [‘grid’, ‘init’, ‘restart’] (default grid)

Default: “grid”

--file

Input file name (full name or root)

--output

Output file name (full name)

--propnames

List of propnames, separate either with spaces or colon

--dates

List of dates, separated by spaces or colon, or filename pointing to a file with one date pr. line.

--standardfmu

Use standard fmu name setting of file (no args)

Default: False

--version

show program’s version number and exit
