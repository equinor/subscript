RESTARTTHINNER¶

Slice a subset of restart-dates from an E100 Restart file (UNRST)

Example:

$ restartthinner --restarts 4 ECLIPSE.UNRST

where four restarts evenly spread out in relevant dates will be picked and written to the same filename (keeping the original is optional)

usage: restartthinner [-h] [-n RESTARTS] [-d] [-q] [-k] [--version] UNRST

Positional Arguments¶

UNRST

Name of UNRST file

Named Arguments¶

-n, --restarts

Number of restart dates wanted

Default: 0

-d, --dryrun

Dry-run only, do not touch files

Default: False

-q, --quiet

Mute output from script

Default: False

-k, --keep

Keep original UNRST file

Default: False

--version

show program’s version number and exit
